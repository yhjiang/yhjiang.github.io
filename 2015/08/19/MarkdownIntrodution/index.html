Markdown语法十分钟入门

Posted by Yinghui Jiang on 2015-08-19

简介

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。 —— 维基百科

工具

  1. Mac系统下推荐使用Mou , 免费版，支持实时预览，左边编写Markdown语言，右边可以实时预览效果。还可以修改主题(Themes)与样式(Css)。

  1. Ulysses –Markdown 编辑器佼佼者，收费版。

    • 支持多种写作格式。
    • 多文档支持。 Ulysses III支持Folder、Filter管理，这一点非常不错。一个Folder里面可以有多个Sheet，并且Sheet之间可以Combine处理。
    • iCloud,Dropbox支持。 支持同步到iCloud Dropbox
    • 支持附件 。在Markdown支持的基础上，可以将附件文件直接拖入文档，建立链接。
    • 多种格式输出。 支持Markdown，RTF，PDF，ePub，HTML，Doc文档输出。
    • 界面舒服。 界面看起来舒服，支持全屏模式。

  1. 马克飞象，专为印象笔记打造的Markdown编辑器。通过精心的设计与技术实现，配合印 象笔记强大的存储和同步功能，带来前所未有的书写体验。特点概述：

    • 功能丰富 ：支持高亮代码块、LaTex 公式、流程图，本地图片上传甚至截图粘贴，工作学习好帮手；
    • 得心应手 ：简洁高效的编辑器，提供桌面离线客户端，支持移动端 Web；
    • 深度整合 ：支持选择笔记本和添加标签，支持从印象笔记跳转编辑，轻松管理。

标题

使用 # 前缀可以设置标题的的样式， 总共有6种标题样式。 # 与文字之间最好空一格。

引用

使用 > (键盘上的大于号)符号在引用的文字前，即可。 记得符号与文字之间空一格

粗体与斜体

使用 两个 * 号 把文字包括起来，就是粗体。

使用 一个 * 号 把文字包括起来，就是斜体。

分割线

使用 三个 * 号 就是分割线。 比如

我是伟大的分割线

有序列表&无序列表

图片与链接

插入图片跟插入链接的语法比较相似，区别是：插入图片要在前面加入一个感叹号 ！

![](这里写入图片地址)

[这里填写标题]（这里填写网址)

引用代码

int main(int argc, char * argv[]){ @autoreleasepool { return UIApplicationMain(argc, argv, nil, NSStringFromClass([AppDelegate class])); } }

使用 ` 符号（就是键盘上Tab键上面那个按钮）把代码包括起来即可。

使用 tab 键即可缩进。
